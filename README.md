📖 FB2Reader — читалка FB2 с анимацией страниц
Элегантная читалка электронных книг в формате FB2 с плавной анимацией перелистывания, ночным режимом, закладками и полной настройкой внешнего вида.
Реализована на 7 языках программирования для демонстрации различных подходов к созданию интерактивных приложений с анимацией.

🧠 Концепция
FB2Reader — это не просто читалка. Это инструмент для максимально комфортного чтения с эффектом реальной книги:

✅ Открытие FB2-файлов — парсинг структуры и извлечение текста с сохранением абзацев.

✅ Анимация перелистывания страниц — плавный переход между страницами с эффектом поворота (имитация бумажной книги).

✅ Ночной режим — переключение между светлой и тёмной темой с сохранением настроек.

✅ Настройка шрифта — размер, тип, межстрочный интервал (в GUI-версиях).

✅ Закладки — сохранение позиции чтения с возможностью быстрого перехода.

✅ Поиск по тексту — с подсветкой найденных фрагментов.

✅ Прогресс чтения — отображение прочитанной части в процентах и в виде полосы.

✅ Автосохранение — восстановление последней открытой страницы после перезапуска.

✅ Горячие клавиши — управление без мыши (пробел/стрелки для перелистывания).

🚀 Как запустить
Для версий с GUI требуется установка соответствующих библиотек. Консольные версии (Go, Rust) работают в терминале и используют ANSI-цвета для ночного режима.

bash
# Python (Tkinter)
pip install tk lxml
python fb2reader_python.py

# C++ (Qt)
qmake && make
./fb2reader_cpp

# Java (Swing)
javac -cp . fb2reader_java.java && java fb2reader_java

# C# (WPF)
dotnet add package System.IO.Compression
dotnet run

# Go (консоль с цветами)
go mod tidy
go run fb2reader_go.go

# Rust (консоль с termion)
cargo build --release && ./target/release/fb2reader_rs

# JavaScript (Node.js + readline)
npm install xml2js
node fb2reader_js.js
